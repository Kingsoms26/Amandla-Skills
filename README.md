Amandla Skills
Amandla Skills is a C2C digital marketplace designed to connect community members with local tradespeople in South Africa. 
The platform aims to digitize the informal township economy by providing a secure environment to find services, request quotes, and manage payments.

Core Features
Role-based dashboards for Clients, Service Providers, and Administrators.
Custom escrow payment system to ensure secure transactions.
Bilingual interface supporting English and IsiXhosa.
Built-in dispute resolution and reporting system.

Technologies Used
Frontend: HTML, CSS (Bootstrap), JavaScript
Backend: PHP
Database: MySQL

Local Setup
To run this project locally, import the provided SQL file into your MySQL database and update the config.php file with your local database credentials. 
Host the project files on a local server environment such as XAMPP.
